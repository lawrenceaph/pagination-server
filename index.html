Visit /api to get the data
Query Params:
page: Starting page (default:1)
perPage: Objects to return per page (default: 10)
Example: https://pagination-server-five.vercel.app/api?page=3&perPage=2
Code above returns:
{"objects":[{"name":"random name 5","content":"random content 5"},{"name":"random name 6","content":"random content 6"}],"nextPage":4}
